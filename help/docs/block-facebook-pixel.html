Facebook Pixel cho Chatbot

Trong bài viết này, chúng tôi sẽ giới thiệu cách thiết lập và cài đặt Facebook Pixel vào AhaChat để chạy quảng cáo Click-to-Messenger và theo dõi chuyển đổi trên FB Ads Manager.

Nếu bạn đã tạo Pixel thì hãy hãy bỏ qua bước 1 và đi đến bước 2 để lấy Token API chuyển đổi.

Thuật ngữ và định nghĩa#

Pixel: Một nguồn dữ liệu từ FB để thu thập tín hiệu chuyển đổi từ những nguồn bên ngoài. Những dữ liệu này được gọi là "Events".

Event (sự kiện): Thông tin mô tả các hành động của khách hàng (người đăng ký). Có 3 loại Event:

  • Standard purchase
  • Standard non-purchase event
  • Custom non-purchase event

Conversion (chuyển đổi): Được hiểu một cách đơn giản là chuyển từ thực thể ở trạng thái A chuyển sang B. Là tỷ lệ giữa tổng số người dùng click vào Quảng Cáo dựa trên một mục tiêu cụ thể nào đó. Có thể, họ truy cập vào để tìm kiếm thông tin sản phẩm, đặt hàng, thanh toán, hay đăng ký thành viên,… Tùy theo lĩnh vực hoạt động khác nhau của từng fanpage mà chúng ta có thể đặt ra mục tiêu và tăng tỷ lệ chuyển đổi theo mục tiêu đó.

Ví dụ cụ thể:

Một Fanpage chuyên kinh doanh đồ chơi trẻ em, Thống kê chiến dịch chạy quảng cáo Click to Messenger vừa rồi có 650 lượt nhắn tin, trong đó có 270 lượt đăng ký mua hàng. Theo đó chúng ta có thể tính ra được tỉ lệ chuyển đổi bằng:

CR = 270/650*100% = 41%

Các bước để tích hợp Facebook Pixel Vào AhaChat#

Bước 1: Tạo Facebook Pixel#

Đi đến Trình quản lý sự kiện https://www.facebook.com/events_manager2

Nhấp vào Kết nối với nguồn dữ liệu và đặt tên cho pixel → bấm tạo.

Bỏ qua mọi thứ liên quan đến cài đặt trang web, URL trang web, v.v. - chúng ta không cần bất kỳ điều gì trong số đó. Chỉ cần tạo Pixel nếu bạn chưa có (bấm dấu X để tắt sau khi đặt tên).

Bây giờ bạn có một ID Pixel mà bạn có thể lưu lại để kết nối vào AhaChat.

Bước 2: Tạo Token truy cập API chuyển đổi#

Đầu tiên cần lưu ý quan trọng là phải chia sẻ pixel của bạn với Tài khoản quảng cáo mà bạn sẽ sử dụng để tạo chiến dịch chuyển đổi. Tài khoản quảng cáo của bạn cần đảm bảo có đầy đủ quyền truy cập vào Pixel.

Các bước thực hiện: Vào Cài đặt trên pixel → mục chia sẻ → chọn chia sẻ với tài khoản quảng cáo

Nhấp vào "Thêm Người" và thêm tài khoản của mình vào Pixel này cấp tất cả các quyền cần thiết.

Bây giờ là lúc để tạo token truy cập API chuyển đổi. Để làm điều đó, hãy tìm Pixel của bạn trong Trình quản lý Sự kiện, vào Cài đặt → kéo xuống mục API Chuyển đổi -> Bấm Tạo mã truy cập.

Sao khi bấm tạo mã sẽ có một chuỗi mã dài mà bạn cần, lưu nó lại và qua bước tiếp theo

Bước 3: Tích hợp Facebook Pixel vào AhaChat#

Trong giao diện của AhaChat bạn vào Cài đặt → tích hợp → kéo xuống tìm mục Tích hợp Facebook Pixel

Dán Pixel ID và Pixel Token mà bạn đã có ở trên vào sau đó bấm lưu lại.

Bước 4: Tạo Event để thu thập chuyển đổi trong kịch bản#

Để tạo block Event thì trong kịch bản bạn vào Tạo mới block → Chọn block Facebook Pixel

Tùy vào mục đích muốn thu thập chuyển đổi mà bạn cài đặt hành động sự kiện pixel thành bất kỳ hành động nào bạn muốn

Nếu mọi thứ được thực hiện chính xác, bạn sẽ có thể xem được kết quả trong báo cáo hiệu suất của Trình quản lý quảng cáo của bạn.
