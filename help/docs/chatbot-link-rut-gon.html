Chatbot tạo link rút gọn

Nếu bạn đang xây dựng hệ thống Affiliate bằng chatbot và mong muốn cung cấp cho các publisher của mình những đường link gọn gàng, đẹp đẽ thì hãy sử dụng dịch vụ Rút Gọn Link Với Domain Riêng aha.chat

Demo: https://m.me/ahachatdemo?ref=rutgonlink

Bước 1: Đăng ký tài khoản tại aha.chat#

Bước 2: Đăng nhập vào lấy API TOKEN để sử dụng trong chatbot#

  • Sau khi đăng nhập bạn di chuyển tới Nhà phát triển → chọn Create token

  • Đặt tên cho Token và bấm Tạo

  • Tạo thành công sẽ xuất hiện API TOKEN như bên dưới, hãy nhớ lưu nó lại vào nơi nào đó vì bạn không thể xem lại token này.

Nếu bạn hiểu về cách dùng API thì đọc thêm hướng dẫn sử dụng tại Rút Gọn Link Với Domain Riêng (API Docs) để có thể rút gọn url theo tên mình mong muốn hay thêm vào UTM tags để theo dõi chiến dịch trên google.

Bước 3: Cấu hình DNS để thêm tên miền của bạn vào https://aha.chat#

3.1 Thông qua trình quản lý tên miền#

  • Truy cập vào trang quản lý tên miền của bạn
  • Thêm bản ghi A này vào cấu hình DNS của bạn:

139.59.108.66

3.2 Thông qua dịch vụ Cloudflare#

  • Đăng ký tài khoản tại https://www.cloudflare.com.
  • Thêm Website vào CloudFlare.

Đầu tiên bạn bấm Add a Site để thêm tên miền của bạn vào hệ thống CloudFlare. Bạn nhập tên miền chính không có www.

  • Lựa chọn Plan.

CloudFlare có khá nhiều plan với các chức năng cao cấp, tuy nhiên bạn có thể Free Plan.

Sau khi chọn Free rồi nhấn Continue, bạn tiếp tục các bước của Cloudflare để xác thực tên miền.

  • Trỏ Name servers về CloudFlare.

CloudFlare cung cấp 2 bản ghi Name servers, bạn sao chép 02 Namesever này thay thế cho 02 Nameserver của bên cung cấp tên miền. Trong trường hợp bạn không biết thao tác thay đổi nameserver trong quản trị tên miền, xin vui lòng liên hệ với bên cung cấp tên miền, nhờ hỗ trợ thay đổi Nameserver về 2 bản ghi Name server của Cloudflare.

  • Tạo bản ghi tên miền về host của aha.chat

Bản ghi cho Tên miền chính

Name: @

Type: A

Content: 139.59.108.66

Bản ghi cho Tên miền phụ

Name: link (là tiền tố phụ mà bạn muốn sử dụng. Tên miền phụ sau đó sẽ có dạng link.domain.com)

Type: A

Content: 139.59.108.66

Bước 4: Thêm tên miền vào aha.chat#

  • Truy cập vào menu Domain Cá Nhân → chọn + Kết nối domain

  • Nhập tên miền chính hoặc phụ của bạn vào Host → bấm Next

  • Bước tiếp theo là hướng dẫn cấu hình DNS bạn đã làm ở bước trên → tiếp tục bấm Next

  • Vậy là xong phần thêm tên miền vào dịch vụ aha.chat. Bạn cần ghi nhớ ID tên miền đã tích hợp của mình để sử dụng trong bước sau.

Bước 5: Đăng nhập và tạo kịch bản trong AhaChat#

  • Bạn copy kịch bản mẫu có chứa sẵn API dưới đây

https://ahachat.com/share-story-to-bot/43bc3bc2c50084e34960625c3008279abKHgPmsjkd3381941

  • Thay đổi API TOKEN

Bạn mở 2 block “api_aha” lên và thêm API TOKEN đã tạo ở Bước 2 vào phần Headers sau Bearer

  • Thay đổi Domain ID

Bạn có thể tìm thấy ID trong giao diện aha.chat

Tiếp tục mở block “api_aha” lên và thêm Domain ID vào phần Data/Body → JSON như ảnh bên dưới.

Note: Nếu không muốn sử dụng link rút gọn bằng tên miền riêng bạn có thể sử dụng link mặc định của aha bằng cách thay giá trị của Domain ID là:

"domain_id": null,

Bài hướng dẫn đã hoàn tất. Xin chúc bạn thành công với các chiến dịch Affiliate của mình bằng link rút gọn ❤
